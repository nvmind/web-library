The Framework

Web Library is a intuitive responsive framework that works out of the box with added features to hep build your next web project quickly.

Download now

Web Library is a resposive, media query driven HTML5 and CSS3 framework. It is designed to be used on any website out of the box. Unlilke some other frameworks, Web Library is designed for mobile devices and desktop. Using CSS media queries the Web Library is able to conform to the screen and shift the content as necessary so that the content is not obscured.

Web Library offers the basic 960 grid with a twist. Using a 12 column container, the framework can size down from 960 to 768, 480, and 320 pixel screen sizes. Web Library does all its work in the browser, and allows for one template to auto-resize to the device without the need to detect devices through either JavaScript or PHP.

Out of the Box

Ideally, Web Library wasn't meant to be the end of a theme, but rather it was meant to be beginning of a better one. Most of the features that exist within Web Library I have personally had to rebuild on most of the themes that I have worked on each time a new theme was created. I started working on Web Library to be the bridge between having to build mobile features into a theme, or even to have to build basic features, such as buttons, and text formats. Out of the box, Web Library is meant to speed up the development process.
